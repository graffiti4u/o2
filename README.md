  uglify-js 팩키지 활용 예.
  npm install uglify-js -g
  npm install uglify-js
  : 글로벌과 로컬의 차이점 확인.
  pretty.js를 만든 후 터미널에서 uglifyjs pretty.js 로 어떻게 바꿨는지 확인해 볼 것.
  -m 플러그를 붙인 후 다시 확인.
  -o 플러그를 추가 붙인 후 다시 확인.
  uglifyjs pretty.js -o uglified.js -m 를 써서 파일을 생성해 본다.
  uglifyjs pretty.js -o pretty.min.js -m 를 써서 min 파일을 생성해 본다.
  : bootstrap과 jQuery 사이트에서 min 파일 확인.
